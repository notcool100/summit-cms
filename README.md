# summit-cms

Eight-page marketing site build (home, about, capabilities, careers, contact, industries, projects, project detail).

## Current structure

- `frontend/` — SvelteKit + TypeScript implementation of the site (see `frontend/README.md`). This is the actively developed version.
- `home.dc.html`, `about.dc.html`, `capabilities.dc.html`, `careers.dc.html`, `contact.dc.html`, `industries.dc.html`, `projects.dc.html`, `project-phoenix.dc.html` — original static HTML/CSS/JS prototypes, kept for reference
- `SiteHeader.dc.html`, `SiteFooter.dc.html` — shared header/footer partials for the static prototypes
- `site.js`, `support.js`, `image-slot.js` — supporting scripts for the static prototypes

## Roadmap

Planned future stack:

- **Backend:** .NET
- **Frontend:** SvelteKit (TypeScript) — see `frontend/`
